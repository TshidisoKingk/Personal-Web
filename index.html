Portfolio

Transactional Fraud Detection System Using Machine Learning Algorithms

This project focuses on building a robust fraud detection system that leverages machine learning algorithms to identify and prevent fraudulent financial transactions. Using a combination of supervised and unsupervised learning techniques—including Random Forest, Support Vector Machines, Logistic Regression, Isolation Forest, and Autoencoders—the system analyzes transaction patterns to detect anomalies in real-time. The model is trained and evaluated using industry-relevant metrics such as accuracy, ROC-AUC, precision-recall score, and confusion matrix. This solution aims to enhance financial security and reduce fraud-related losses for institutions.

SizaTelco99: Telecom Fraud Detection System

SizaTelco99 is an advanced hybrid fraud detection system developed to combat prepaid and postpaid fraud in the telecom industry. It integrates machine learning, explainable AI (XAI), and big data analytics to accurately detect suspicious activities in real-time. The system employs supervised and unsupervised models to analyze call records, billing data, and user behavior patterns. Designed with scalability and transparency in mind, SizaTelco99 not only identifies fraud but also provides clear, explainable insights to aid telecom operators in understanding and mitigating emerging threats.
